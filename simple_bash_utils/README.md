# Simple bash utils
В данном проекте предстояло разработать свою реализацию unix-утилит grep и cat.

## cat
### Сборка и запуск
```
make s21_cat
./s21_cat test1.txt
```
### Тесты
```
make test
```
## grep
### Сборка и запуск
```
make s21_grep
./s21_grep grep_test1.txt "comp"
```
### Тесты
```
make test
```
