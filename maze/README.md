# Maze Pathfinder

Программа для генерации лабиринтов и поиска кратчайшего пути между двумя точками с использованием текстового интерфейса (ncurses).

## Функциональность

### Управление лабиринтом

| Операция | Описание | Исключительные ситуации |
|----------|----------|-------------------------|
| `generate_maze(maze, rows, cols)` | Генерирует случайный лабиринт заданного размера. | Некорректный размер (rows ≤ 0, cols ≤ 0, rows > MAX_ROWS, cols > MAX_COLS). |
| `read_maze(maze, filename)` | Загружает лабиринт из файла. | Файл не существует или имеет неверный формат. |
| `save_maze_to_file(maze, filename)` | Сохраняет лабиринт в файл. | Некорректный путь к файлу, ошибка записи. |
| `destroy_maze(maze)` | Освобождает память, занятую лабиринтом. | Передан NULL-указатель. |

### Поиск пути

| Операция | Описание | Исключительные ситуации |
|----------|----------|-------------------------|
| `find_path_in_maze(maze, start_i, start_j, finish_i, finish_j)` | Находит кратчайший путь между двумя точками с использованием BFS. | Некорректные координаты (выходят за границы лабиринта). |
| `create_image_maze(maze, maze_image)` | Создает текстовое представление лабиринта с отображением пути. | Ошибка выделения памяти. |
| `destroy_maze_image(maze_image)` | Освобождает память, занятую изображением лабиринта. | Передан NULL-указатель. |

### Пользовательский интерфейс

| Функция | Описание |
|---------|----------|
| `init_curses()` | Инициализация ncurses и цветовых схем. |
| `select_maze_source(choice)` | Выбор источника лабиринта: файл или генерация. |
| `select_maze_size(rows, cols)` | Ввод размера лабиринта. |
| `select_start_and_finish(start, finish)` | Ввод координат начальной и конечной точек. |
| `draw_maze_image(maze_image)` | Отрисовка лабиринта с цветовой разметкой. |
| `show_error(error)` | Отображение сообщения об ошибке. |
| `show_goodbye()` | Экран прощания. |
| `stop_curses()` | Завершение работы с ncurses. |

### Структуры данных

| Структура | Описание |
|-----------|----------|
| `maze_t` | Основная структура лабиринта: размеры, матрицы правых и нижних стен, матрица пути. |
| `maze_image_t` | Текстовое представление лабиринта: строки, столбцы, массив символов. |
| `queue_t` | Очередь для BFS: указатели на начало и конец, размер. |
| `Point` | Координаты ячейки лабиринта (строка, колонка). |
