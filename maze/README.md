# Maze Pathfinder

Программа для генерации лабиринтов и поиска кратчайшего пути с использованием текстового интерфейса (ncurses).

В проекте реализован алгоритм генерации лабиринтов на основе метода Эллера и поиск кратчайшего пути с помощью BFS (поиска в ширину). Интерфейс построен с использованием библиотеки ncurses для цветного отображения ASCII-графики.

## Функциональность

### Управление лабиринтом

| Операция | Описание | Исключительные ситуации |
|----------|----------|-------------------------|
| `generate_maze(maze, rows, cols)` | Генерирует случайный лабиринт заданного размера. | Некорректный размер (rows ≤ 0, cols ≤ 0, rows > MAX_ROWS, cols > MAX_COLS) |
| `read_maze(maze, filename)` | Загружает лабиринт из файла. | Файл не существует или имеет неверный формат |
| `save_maze_to_file(maze, filename)` | Сохраняет лабиринт в файл. | Некорректный путь к файлу, ошибка записи |
| `destroy_maze(maze)` | Освобождает память, занятую лабиринтом. | Передан NULL-указатель |

### Поиск пути

| Операция | Описание | Исключительные ситуации |
|----------|----------|-------------------------|
| `find_path_in_maze(maze, start_i, start_j, finish_i, finish_j)` | Находит кратчайший путь между двумя точками с использованием BFS. | Некорректные координаты (выходят за границы лабиринта), путь не существует |
| `create_image_maze(maze, maze_image)` | Создает текстовое представление лабиринта с отображением пути. | Ошибка выделения памяти |
| `destroy_maze_image(maze_image)` | Освобождает память, занятую изображением лабиринта. | Передан NULL-указатель |

### Пользовательский интерфейс

| Функция | Описание |
|---------|----------|
| `init_curses()` | Инициализация ncurses и цветовых схем. |
| `stop_curses()` | Завершение работы с ncurses. |
| `select_maze_source(choice)` | Выбор источника лабиринта: файл или генерация. |
| `select_maze_size(rows, cols)` | Ввод размера лабиринта. |
| `select_start_and_finish(start, finish)` | Ввод координат начальной и конечной точек. |
| `draw_maze_image(maze_image)` | Отрисовка лабиринта с цветовой разметкой. |
| `show_error(error)` | Отображение сообщения об ошибке. |
| `show_goodbye()` | Экран прощания. |
| `get_maze_filename(filename, max_len)` | Запрос имени файла у пользователя. |

### Структуры данных

| Структура | Описание |
|-----------|----------|
| `maze_t` | Основная структура лабиринта: размеры, матрицы правых и нижних стен, матрица пути |
| `maze_image_t` | Текстовое представление лабиринта: строки, столбцы, массив символов |
| `queue_t` | Очередь для BFS: указатели на начало и конец, размер |
| `Point` | Координаты ячейки лабиринта (строка, колонка) |

## Поток работы программы

1. **Инициализация** — запуск ncurses и цветовых схем
2. **Выбор источника лабиринта**:
   - `[1]` — загрузить из файла
   - `[2]` — сгенерировать случайный
   - Любая другая клавиша — выход
3. **При загрузке из файла** — ввод имени файла
4. **При генерации**:
   - Ввод размера лабиринта (rows cols)
   - Ввод имени файла для сохранения
5. **Ввод координат** — начальной и конечной точек (row col)
6. **Отображение** — лабиринт с найденным путем (отмечен символом `*`)
7. **Завершение** — экран прощания, очистка памяти

## Формат файла лабиринта

```
rows cols
right_walls[0][0] right_walls[0][1] ... right_walls[0][cols-1]
...
right_walls[rows-1][0] ... right_walls[rows-1][cols-1]
bottom_walls[0][0] bottom_walls[0][1] ... bottom_walls[0][cols-1]
...
bottom_walls[rows-1][0] ... bottom_walls[rows-1][cols-1]
```

Где `1` означает наличие стены, `0` — отсутствие.

**Пример файла 3×3:**
```
3 3
0 0 1
1 0 0
0 0 1
1 0 1
0 0 0
1 1 1
```

## Коды ошибок

| Код | Описание |
|-----|----------|
| `NO_ERR` | Ошибок нет |
| `MEM_ERR` | Ошибка выделения памяти |
| `FILE_ERR` | Ошибка работы с файлом |
| `PARAM_ERR` | Некорректные параметры (размер, координаты) |

## Ограничения

- Максимальный размер лабиринта: **50×50** (определено константами MAX_ROWS и MAX_COLS)
- Поддерживаются только прямоугольные лабиринты
- Для работы требуется библиотека **ncurses**
- Интерфейс только текстовый (терминал)

## Управление памятью

| Функция | Описание |
|---------|----------|
| `allocate_maze(maze, rows, cols)` | Выделяет память под структуру лабиринта |
| `destroy_maze(maze)` | Освобождает память лабиринта |

## Цветовая схема

| Элемент | Цвет | Описание |
|---------|------|----------|
| Стены (`+`, `-`, `\|`) | Зеленый | Границы лабиринта |
| Пустое пространство | Черный | Проходимые ячейки |
| Путь (`*`) | Красный | Кратчайший путь от старта до финиша |
| Текст интерфейса | Синий | Меню, подсказки, сообщения |
| Сообщения об ошибках | Красный | Вывод ошибок |
