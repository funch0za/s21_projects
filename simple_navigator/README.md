Программа для работы с графами: поиск путей, минимальное остовное дерево и решение задачи коммивояжёра с использованием текстового интерфейса.

В проекте реализованы основные алгоритмы на графах: DFS и BFS обходы, алгоритмы Дейкстры и Флойда-Уоршелла для поиска кратчайших путей, алгоритм Прима для построения минимального остовного дерева, а также муравьиный алгоритм для приближенного решения задачи коммивояжёра.

## Зависимости

* **Check** — для модульного тестирования
* **gcovr** — для генерации отчета о покрытии кода

## Сборка

```bash
make all
```

Для сборки отдельных компонентов:

```bash
make s21_graph          # Только библиотека графа
make s21_graph_algorithms # Только библиотека алгоритмов
make s21_navigator      # Сборка исполняемого файла
```

## Запуск

```bash
./s21_navigator
```

При запуске программа запросит имя файла с графом. Примеры файлов находятся в директории `examples/`.

## Формат входного файла

```
размер
матрица_смежности[0][0] матрица_смежности[0][1] ... матрица_смежности[0][n-1]
...
матрица_смежности[n-1][0] матрица_смежности[n-1][1] ... матрица_смежности[n-1][n-1]
```

Где `0` означает отсутствие ребра, положительное число — вес ребра.

**Пример файла (3 вершины):**
```
3
0 5 3
5 0 4
3 4 0
```

## Вывод программы

При успешной загрузке графа программа последовательно выполняет:

1. **DFS обход** — для каждой вершины
2. **BFS обход** — для каждой вершины
3. **Алгоритм Дейкстры** — кратчайший путь между двумя случайными вершинами
4. **Алгоритм Флойда-Уоршелла** — кратчайшие пути между всеми парами вершин
5. **Алгоритм Прима** — минимальное остовное дерево
6. **Муравьиный алгоритм** — решение задачи коммивояжёра
